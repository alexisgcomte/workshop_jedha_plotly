# Workshop Jedha Plotly

## I. Installation

### 1. Python env installation

```bash
python3 -m venv env
source env/bin/activate
pip3 install --upgrade pip
pip install -r requirements.txt
```
### 2. Start Jupyter Server

```bash
jupyter notebook
```

### 3. Start Streamlit Server 

```bash
streamlit run streamlit/app.py
```
